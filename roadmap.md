## 📌 Reality Check

- Degree is helpful but **not mandatory** in many cybersecurity roles.
- Certifications help:
  - Pass HR filters
  - Show commitment
  - Provide structured learning path
- Technical interviewers often care more about:
  - Labs
  - Linux skill
  - Automation skill
  - Security understanding
  - Problem solving

# 🛡️ Major Cybersecurity Certification Providers

---

## 🥇 Core Industry Certification Organizations

### ISC2 (International Information System Security Certification Consortium)
**Famous Certifications**
- CISSP
- CCSP
- SSCP
- CC (Entry Level)

**Why It Matters**
- One of the most globally recognized security organizations
- CISSP = Senior / architecture / leadership roles
- CC = Good entry point (sometimes free programs available)

---

### CompTIA
**Famous Certifications**
- Security+
- CySA+
- PenTest+
- CASP+

**Why It Matters**
- Most common first security certification
- Vendor neutral → Good for career switchers

---

### EC-Council
**Famous Certifications**
- CEH (Certified Ethical Hacker)
- CHFI
- CND
- CCISO

**Why It Matters**
- Very strong HR keyword recognition
- CEH is widely known (even outside technical teams)

---

### SANS / GIAC
**Famous Certifications**
- GSEC
- GPEN
- GCIH
- GCIA

**Why It Matters**
- Very technical
- Highly respected by engineers
- Expensive (usually later career)

---

### Offensive Security (OffSec)
**Famous Certifications**
- OSCP
- OSWE
- OSEP

**Why It Matters**
- Offensive security gold standard reputation
- Very practical and skill-focused

# Free IT Certifications & Training Guide

This guide highlights reputable organizations offering free training and certifications in Cybersecurity, Networking, and Linux.

## 🛡️ Cybersecurity
| Organization | Certification/Course | Best For |
| :--- | :--- | :--- |
| **ISC²** | Certified in Cybersecurity (CC) | **Top Pick:** Free training and exam. |
| **Cisco** | Intro to Cybersecurity | Foundational security concepts. |
| **Fortinet** | NSE 1 & 2 | Threat landscapes and security basics. |
| **Palo Alto** | PCCET | Entry-level security fundamentals. |

## 🌐 Networking
| Organization | Certification/Course | Best For |
| :--- | :--- | :--- |
| **Cisco** | Networking Essentials | Basic networking knowledge and labs. |
| **Microsoft** | Azure Fundamentals (AZ-900) | Cloud networking concepts. |

## 🐧 Linux
| Organization | Certification/Course | Best For |
| :--- | :--- | :--- |
| **Linux Foundation** | Intro to Linux (edX) | Foundational Linux skills. |
| **Cisco** | NDG Linux Essentials | Entry-level admin preparation. |
| **Red Hat** | RHEL Technical Overview | Red Hat platform fundamentals. |

---

## 📝 Important Notes
* **Training vs. Exam:** Many platforms offer free training, but the official proctored exam may have a fee.
* **Certificates of Completion:** These are still highly valuable for resumes and LinkedIn to prove you completed the coursework.
* **Top Recommendation:** The **ISC² Certified in Cybersecurity (CC)** is currently the best option for a 100% free official certification (training + exam).
